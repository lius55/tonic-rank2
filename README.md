# tonic-rank2
一个洗发水内容网站的动态化。  
其实并没有太多技术含量，只是单纯的用ajax方式调用了一下API，然后把返回值绑定到html页面中。  
API的返回值跟页面的绑定使用了jquery template.  

## Keyword
- html
- jquery
- jquery template
- php
